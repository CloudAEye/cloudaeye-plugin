This section allows you configure a POST webhook to export your jenkins job logs and relevant metadata to CloudAEye to perform Root Cause Analysis on Test Failures. This plugin is supported for both freestyle and pipleline projects.

To enable this plugin at individual build project level:

  • FreeStyle Project: To enable this plugin for a selected free-style project, you need to enable this a post-build action step.
  • Pipeline Project: To enable this plugin for a selected pipeline project, you need to add a new step to the post section in your pipeline script. this a post-build action step.
